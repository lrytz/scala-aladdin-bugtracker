subject [nsc] (bug of the year) Erasure 'erases' useful expressions
code 
class Foo extends AnyRef with Application {
  var a = Predef.Array(1, 2, 4, 5);

  def swap(i: Int, j: Int): Unit = {
    val t = a(i); 
    a(i) = a(j); 
    a(j) = t // this will disappear!!!!
  }

  swap(2, 3);
}
what happened 
The last assignment in method swap evaporates during erasure phase! It's weird because another expression, for e\
xample a call to println will not dissapear!
what expected To stay there! :-)
