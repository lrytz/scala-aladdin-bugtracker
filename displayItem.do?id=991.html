		what happened
		
Plugin does not provide option to use real tabs insteadof of spaces.


	

	

		what expected
		I would really appreciate it if the Scala Eclipse Plugin had
an option to use real tabs instead of spaces.

By making it configurable how wide a tab has to be when
displayed you can achieve the default of 2 spaces we have now.

Moreover in normal Eclipse key-bindings mode it is easier to
navigate indenting levels when real tabs are used.

As far as I'm concerned it is a purely subjective personal preference
how wide indenting levels should be and using tabs allows that.
